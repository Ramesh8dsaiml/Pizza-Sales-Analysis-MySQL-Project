# Pizza-Sales-Analysis-MySQL-Project
SQL-based analysis project using MySQL to explore pizza sales, revenue trends, customer ordering patterns, and category-wise insights. Includes basic, intermediate, and advanced SQL analytical queries.

This project contains SQL-based analysis performed on a Pizza Sales dataset using MySQL.
The goal is to explore sales performance, revenue trends, order patterns, and category-wise insights using SQL.
